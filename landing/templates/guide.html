📖 Как подключить RadarShield

Выбери свою платформу — инструкция откроется ниже.

FLClash для Android
APK · бесплатно, без аккаунта
Скачать
1

Установка и первый запуск

При первом запуске появятся два системных диалога:

  • «Отказ от ответственности» — нажми Согласен
  • «Уведомление о сборе данных» (Firebase Crashlytics) — нажми Подтвердить
2

Добавить профиль

  1. Перейди во вкладку Профили (нижняя панель)
  2. Нажми + Добавить профиль
  3. Выбери URL → откроется диалог «Импорт из URL»
  4. Вставь свою ссылку-подписку RadarShield
  5. Нажми Отправить

После загрузки в списке появится профиль RadarShield с меткой «Только что».

💡 Ссылку подписки можно получить в боте командой /profile
3

Подключиться

  1. Перейди на вкладку Панель управления
  2. Нажми кнопку подключения (значок ▶ вверху)
  3. Появится запрос «FlClash пытается подключиться к сети VPN» — нажми OK
  4. Статус сменится на зелёную галочку ✅ — VPN активен
✓

Проверка

На главном экране в блоке «Обнаруж...» должен появиться IP-адрес (не российский). Трафик ↑/↓ начнёт расти — значит всё работает.

Режимы исходящего трафика

РежимОписание
ПравилоРекомендуется — заблокированное через VPN, остальное напрямую
ГлобальныйВесь трафик через VPN
ПрямойВесь трафик без VPN

По умолчанию выбрано Правило — оставь как есть.

Karing для iOS
App Store · iPhone / iPad
Скачать
1

Первый запуск

Появится экран «Your Privacy Comes First» — Karing собирает модель устройства и анонимные логи. Данные третьим лицам не передаются.

Нажми Accept & Continue.

2

Начальная настройка

  • Язык → выбери Русский → нажми Дальше
  • Страна или регион → выбери RU — Российская Федерация → нажми Дальше
  • Режим новичка — оставь включённым на первое время
⚠️ Правильно укажи текущую страну — иначе возможны проблемы с перенаправлением в сети.
3

Добавить подписку

  1. Нажми «+ Добавить профи…» на главном экране
  2. Выбери «Добавление подписки»
  3. В поле «Ссылка на подписку/содержание» вставь свою ссылку RadarShield
  4. Нажми галочку ✓ в правом верхнем углу
💡 Ссылку подписки можно получить в боте командой /profile
4

Разрешить VPN

iOS покажет диалог: «Karing запрашивает разрешение на добавление конфигураций VPN»

Нажми Разрешить (серая кнопка внизу).

5

Подключиться

На главном экране нажми кнопку «Соединить». Статус изменится на «Подключено».

✓

Готово

Текущая конфигурация отображается как «Автовыбор» — приложение само выберет лучший сервер.

1

Скачай и установи клиент

На Windows — FLClash: запусти .exe. На Mac — Karing из App Store: то же приложение, что и на iPhone, ставится в один клик.

Нужен FLClash на Mac (запасной вариант)

Скачай версию для M-серии или для Intel — архитектуру видно в «Об этом Mac».

При первом запуске macOS напишет, что «не может проверить приложение на наличие вредоносного ПО». Так система относится ко всем программам не из App Store, с самим VPN это не связано. Открыть можно двумя способами:

  • Системные настройки → Конфиденциальность и безопасность → пролистай вниз → «Всё равно открыть».
  • Или в Терминале: xattr -dr com.apple.quarantine /Applications/FlClash.app — после этого приложение открывается обычным двойным кликом.
2

Добавь профиль по URL

  1. Открой клиент → нажми + → Import from URL (в Karing — Импорт из буфера обмена)
  2. Вставь свою ссылку-подписку RadarShield
  3. Нажми Подключить
💡 Ссылку подписки можно получить в боте командой /profile. Или отсканируй QR-код на странице профиля.
3

Подключиться

На главном экране нажми кнопку подключения. Статус сменится на Connected.

Режим Правило выбирается автоматически — он оптимален для большинства задач.
