1. Передача платёжных данных

Ввод данных банковской карты (номер, срок действия, CVC/CVV) происходит исключительно на стороне платёжного сервиса Robokassa (ООО «РОБОКАССА», ОГРН 1055009302215, ИНН 5047063929 — банковский платёжный агент платёжного агрегатора), сертифицированного по стандарту PCI DSS. Оператор Сервиса {{ ip_full_name }} не получает, не хранит и не передаёт данные карт.

2. Защита соединения с сайтом

Весь трафик между браузером Пользователя и нашими серверами защищён протоколом TLS версии 1.2 и выше с сертификатом, выпущенным центром сертификации Let's Encrypt. Использование устаревших протоколов (SSL 3.0, TLS 1.0 / 1.1) отключено.

3. 3-D Secure

Все платежи проходят дополнительную аутентификацию по технологии 3-D Secure (Verified by Visa, Mastercard SecureCode, МИР Accept) при поддержке банка-эмитента карты Пользователя.

4. Защита инфраструктуры

  • Серверы Сервиса физически расположены в дата-центрах с контролем доступа и резервированием питания/связи;
  • Доступ администратора — только по SSH с ключами (2048+ бит), пароли отключены;
  • Внешний периметр защищён файрволом (UFW), открыты только необходимые порты;
  • Автоматическое обновление критических патчей безопасности операционной системы;
  • Ежедневное резервное копирование базы данных в защищённом виде.

5. Защита аккаунта Пользователя

Персональная ссылка-подписка (subscription URL) содержит защищённый HMAC-токен, проверяемый на стороне сервера. В случае утечки ссылки Пользователь может в любой момент написать в поддержку для перевыпуска учётных данных (при этом старая ссылка мгновенно перестаёт работать).

6. Что делать при подозрении на инцидент

Если вы заметили, что ваш аккаунт использует кто-то другой (например, превышение трафика, которого вы не делали), немедленно напишите в поддержку на {{ support_email }} или в Telegram. Мы заблокируем старые учётные данные и выпустим новую ссылку в течение 24 часов.

7. 152-ФЗ

Полный перечень обрабатываемых персональных данных, цели и сроки хранения описаны в «Политике конфиденциальности». Оператор обрабатывает персональные данные в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».
